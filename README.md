## 🚀 Getting Started

### install
```sh
pnpm install
```
### set environmental values
set .env.local under root dir with values list below
```sh
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
KV_REST_API_URL=
KV_REST_API_TOKEN=
GEMINI_API_KEY=
```
### usege
```sh
pnpm run dev
```
