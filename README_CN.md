## 🚀 快速开始

### 安装依赖
```sh
pnpm install
```
### 设置环境变量
set .env.local under root dir with values list below
```sh
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
KV_REST_API_URL=
KV_REST_API_TOKEN=
GEMINI_API_KEY=
```
### 本地开发
```sh
pnpm run dev
```
